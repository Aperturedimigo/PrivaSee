##Installation
In your repository
####Linux:
```
sudo pip install virtualenv
virtualenv .venv
source .venv/bin/activate
pip install cognitive_face
pip install requests
sudo apt-get install python3-pyqt5
```
####windows:
```
pip install cognitive_face
pip install requests

```

and clone this files and add these code:

```
from Minsung import *
```


##Sample

If you want to request `Detect`:

```
from Minsung import *
```
